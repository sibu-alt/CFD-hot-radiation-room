# CFD_ Hot Radiation Room

The hot Radiation Room problem models heat transfer and fluid flow in an enclosed space where thermal radiation plays a significant role. It is a benchmark case for studying coupled convection, conduction and radiation phenomena in CFD. It captures the interaction between fluid motion, wall conduction and radiative heat transfer. Real-world scenarios such as heated enclosures, furnaces or solar-exposed rooms can be simulated.
